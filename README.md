# 动物园数字孪生交互系统规范

## 1. 项目概述

- **项目名称**: Zoo Digital Twin Interactive System
- **项目类型**: 3D交互式Web应用
- **核心功能**: 基于Three.js构建的动物园三维场景，支持多视角切换、昼夜变换、动物信息展示等交互功能
- **目标用户**: 动物园游客、教育展示、虚拟游览

## 2. UI/UX 规范

### 布局结构

- **主画布**: 全屏3D渲染区域，占据整个视口
- **信息面板**: 右侧半透明浮动面板，显示选中对象信息
- **控制面板**: 底部中央，包含视角切换、昼夜切换等功能按钮
- **标题栏**: 左上角，显示系统名称

### 视觉设计

- **主色调**:
  - 白天模式: 天空蓝 `#87CEEB`，草地绿 `#228B22`
  - 夜间模式: 夜空深蓝 `#1a1a2e`，草地深绿 `#0d3d0d`
- **UI面板**: 半透明深色 `rgba(20, 20, 35, 0.85)`
- **强调色**: 金色 `#FFD700`，用于按钮高亮
- **文字颜色**: 白色 `#FFFFFF`，次要文字 `#AAAAAA`
- **字体**: "Microsoft YaHei", "PingFang SC", sans-serif
- **圆角**: 8px，按钮更圆润

### 组件设计

1. **信息面板**
   - 尺寸: 280px × auto，最大高度400px
   - 包含: 标题、描述文字、图片占位符
   - 展开/收起动画

2. **控制按钮组**
   - 水平排列，间距12px
   - 按钮尺寸: 48px × 48px (图标按钮) / auto × 36px (文字按钮)
   - 悬停效果: 背景变亮，轻微放大

3. **动物指示器**
   - 鼠标悬停时显示名称标签
   - 点击选中时高亮边框

## 3. 功能规范

### 3.1 场景对象（至少3类）

1. **动物模型**（用几何体代替）:
   - 狮子: 橙色球体组合，高约2单位
   - 大象: 灰色椭球体，长约3单位
   - 长颈鹿: 黄色圆柱+球体，高约4单位
   - 斑马: 白色条纹盒子
   - 熊猫: 黑白球体组合

2. **环境装饰**:
   - 石子路: 灰色不规则小石子散落形成的小径
   - 花草植被: 绿色圆锥体（树）+ 彩色小 sphere（花朵）
   - 河流: 蓝色平面，带波纹动画效果
   - 围栏: 木质棕色长方体围成区域

3. **地形**:
   - 草地: 绿色平面，带纹理噪点
   - 山丘: 地形起伏

### 3.2 视角系统（2种）

1. **透视视角**（默认）:
   - OrbitControls 自由旋转
   - 初始位置: (30, 20, 30)
   - 目标点: (0, 0, 0)

2. **鸟瞰视角**:
   - 俯视角度
   - 位置: (0, 60, 0)
   - 可360度旋转查看

### 3.3 交互控制（3类以上）

1. **鼠标控制**:
   - 左键拖拽: 旋转视角
   - 右键拖拽: 平移视角
   - 滚轮: 缩放
   - 点击对象: 选中显示信息

2. **UI控制**:
   - 视角切换按钮
   - 昼夜切换按钮
   - 重置视角按钮

3. **射线检测**:
   - 鼠标悬停动物时显示名称
   - 点击选中动物弹出详细信息面板

### 3.4 核心功能

1. **图文介绍**:
   - 点击动物显示信息面板
   - 包含: 动物名称、学名、描述、栖息地

2. **视角变换**:
   - 平滑过渡动画
   - 保留用户自定义视角

3. **昼夜变换**:
   - 天空颜色渐变
   - 光照强度变化
   - 氛围灯效果

4. **自动漫游**（可选）:
   - 缓慢旋转浏览
